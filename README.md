## URL shortener

User can input any URL and generate a 5-character short code link.

### Environment Setup

* Node.js
* Express.js
* Nodeman
* MongoDB
* Mongoose
* Valid-url

### Installation

1. Clone Repo
``
$ git clone https://github.com/Huntershih/url-shortener.git
``

2. Install
``
$ npm install
``

3. Run seed
``
$ npm run seed
``

4. Run App
``
$ npm run dev
``
